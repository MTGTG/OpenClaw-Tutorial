Openclaw全网最简单安装教学

一、安装WSL和Ubuntu

第一步. 配置 WSL 基础环境

以管理员身份打开 PowerShell，输入以下命令

wsl --install

验证安装状态

安装后，以管理员身份打开 PowerShell，分别输入以下两条命令验证状态，出现版本号即安装成功。(没有出现版本号请重启电脑）

wsl --version
wsl --status

第二步. 输入以下两条命令，启用功能

启用 Windows 内置的 WSL 基础功能

dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart

启用 Windows 底层虚拟化支持

dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart

最后输入以下命令设置为WSL2，然后重启电脑。

wsl --set-default-version 2

第三步. Ubuntu 初始化配置

以管理员身份打开 PowerShell，输入以下命令

wsl --install -d Ubuntu-22.04

安装完成后会自动启动Ubuntu，首次启动 Ubuntu 时，系统会提示创建 UNIX 用户名和密码：

输入你的用户名，然后输入密码，输入密码时是看不见的，实际是有的。输入完成后它会让你重新输入一遍密码，以防设置失败。

检查Ubuntu版本

wsl --list --verbose

第四步. WSL2 的高级网络配置项

打开C盘中的用户，选择正在使用的用户文件夹，鼠标右键创建文本文档，将以下命令复制粘贴到文本中，文件命名为：.wslconfig

[wsl2]
networkingMode=mirrored
dnsTunneling=true
autoProxy=true
firewall=true

[experimental]
autoMemoryReclaim=gradual
hostAddressLoopback=true

然后回到命令行，输入以下两条命令防止端口访问被阻止

New-NetFirewallRule -DisplayName "OpenClaw-Service" -Direction Inbound -Action Allow -Protocol TCP -LocalPort 18789
Get-NetFirewallRule -DisplayName "OpenClaw-Service" | Format-Table

第五步.(可跳过）*此处为危险操作，等于放开Openclaw执行权限，当它执行任何命令式都不再需要密码，风险自担

sudo visudo

在文件末尾添加以下内容（将 前面的xiaolin替换为你的实际用户名）：

你的用户名 ALL=(ALL) NOPASSWD: ALL

举例我的是：
xiaolin ALL=(ALL) NOPASSWD: ALL

保存并退出（Ctrl+O → 回车 → Ctrl+X）

二、安装Openclaw

第一步.安装基础工具

sudo apt install -y curl wget git

第二步. 安装 OpenClaw 官方脚本

curl -fsSL https://openclaw.ai/install.sh | bash

第三步. 安装设置：

  1. 已阅读以上安全条款选择Yes
  1. 快速和手动模式，选择手动模式Manul
  1. 在本地还是远程部署网关服务，选择本地Local gateway
  1. 直接回车，无需修改
  1. 接入大模型API，按各自使用的大模型设置即可。我这里使用的是Deepseek，选择Custom Provider自定义
  1. 将默认API Base URL修改成Deepseek的官方调用API
https://api.deepseek.com/v1
  1. 选择Paste API Key now
  1. 输入大模型的API Key
  1. 使用的是OpenAI的兼容模式，选择第一个
  1. 输入使用的AI模式名字，这里使用的是deepseek-chat，deepseek思考模式为：deepseek-reasoner
  1. 接入社交平台如“飞书WhatsApp/Telegram/”，暂时跳过选择NO
  1. 增加搜索服务，可以获取实时最新信息，也是等后续再进行设置，选择Skip for now
  1. 同样Skills可以等到后面再安装，选择NO
  1. 这个是启用钩子，暂时不设置，选择Skip for now，点击空格，然后回车
  1. 选择Yes
  1. 直接回车
  1. 选择Openclaw在哪里启动，我们选择第二个Web UI,也就是网页界面
  1. 最后选择Yes，设置完成
  1. 设置完成后往上拉动命令行，找到Dashboard ready中的Dashboard link直接复制下面的链接在浏览器中打开。

也可以直接Ctrl+鼠标左键点击这个链接跳转打开Openclaw控制台
